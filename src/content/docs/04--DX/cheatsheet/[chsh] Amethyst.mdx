# Amethyst


[Amethyst](https://ianyh.com/amethyst/) is a tiling window manager for MacOS. This cheat sheet is to support my attempt to remember all of the keyboard shortcuts that are useful for me. See Amethyst preferences for the full list, or to record/remap your own.

---

| Shortcuts                                                                 | Description                       |
| ------------------------------------------------------------------------- | --------------------------------- |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd>                   | Cycle layout forward              |
| <kbd>Ctrl</kbd> + <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | Cycle layout backwards            |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd>                       | Force windows to be reevaluated   |
|                                                                           |                                   |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd>                       | Fullscreen layout                 |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd>                       | Column layout                     |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd>                       | Tall layout                       |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd>                       | Wide layout                       |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd>                       | Display current layout            |
|                                                                           |                                   |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>W</kbd>                       | Focus on Screen 1                 |
| <kbd>Ctrl</kbd> + <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>W</kbd>     | Place focussed window on Screen 1 |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>                       | Focus on Screen 2                 |
| <kbd>Ctrl</kbd> + <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>     | Place focussed window on Screen 2 |
| <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd>                       | Focus on Screen 3                 |
| <kbd>Ctrl</kbd> + <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd>     | Place focussed window on Screen 3 |
|                                                                           |                                   |
| <kbd>Ctrl</kbd> + <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>X</kbd>     | Toggle focus follows mouse        |
| <kbd>Ctrl</kbd> + <kbd>Option</kbd> + <kbd>Shift</kbd> + <kbd>T</kbd>     | Toggle global tiling              |
